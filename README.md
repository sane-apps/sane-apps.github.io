# SaneApps Website

This repository contains the public SaneApps homepage at https://saneapps.com.

The site presents SaneApps as transparent-code, privacy-first macOS utilities built for Mac. Product-specific privacy and support details live in each app repository and website.

## Local Preview

Open `index.html` in a browser for a static preview.

## Deploy

SaneApps websites are deployed through the shared SaneProcess release tooling and Cloudflare Pages. Use the project release scripts instead of deploying this repository by hand.

## Public Messaging

Use these terms consistently:

- `transparent code`
- `100% Transparent Code`
- `source-available` when describing PolyForm Shield app repositories
- `open source` only for repositories with an OSI-approved license

The privacy position is: app data stays on device by default, and SaneApps does not collect personal customer data from product usage.
